# 🌐 موقع شخصي - عبدالله مصطفى

## 📖 نظرة عامة
هذا المشروع عبارة عن موقع شخصي بسيط يعرّف عني ويعرض بعض المعلومات الأساسية عني، مهاراتي، اهتماماتي، وتجربتي الدراسية.  
الموقع مبني باستخدام **HTML, CSS, JavaScript** ويحتوي على صفحات متعددة.

## 🚀 المميزات
- تصميم بخلفية متدرجة جذابة
- صفحات متعددة (عنّي، مهاراتي، المدرسة، كرة القدم)
- نظام تسجيل دخول وتسجيل مستخدمين باستخدام LocalStorage
- إمكانية إضافة وحذف التعليقات
- زر للرجوع إلى أعلى الصفحة عند كثرة المحتوى
- تصميم متجاوب يعمل على مختلف الأجهزة

## 🛠️ التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API

## 🎯 تحسينات مستقبلية
- إضافة المزيد من التنسيقات باستخدام مكتبات مثل Bootstrap أو Material UI
- تحسين التحقق من صحة البيانات في النماذج
- تطوير واجهة المستخدم لتكون أكثر تفاعلية
- نشر المشروع على GitHub Pages أو Netlify
