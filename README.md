# مشروع تجريبي - HTML/CSS/JS

## 📌 الفكرة
هذا المشروع هو بداية لتعلم أساسيات تطوير الويب باستخدام:
- **HTML** لبناء هيكل الصفحة
- **CSS** لتنسيق وتصميم الواجهة
- **JavaScript** لإضافة التفاعلات والوظائف

## 🚀 طريقة التشغيل
1. قم بتحميل المشروع أو نسخه من GitHub.
2. افتح الملف `index.html` في أي متصفح حديث.
3. تأكد أن الملفات (`style.css` و `main.js`) موجودة في نفس المجلد.

## 🛠️ المزايا الحالية
- صفحة رئيسية بسيطة.
- تنسيقات أساسية باستخدام CSS.
- وظائف أولية عبر JavaScript.

## 📅 الخطوات القادمة
- إضافة المزيد من التنسيقات.
- تطوير تفاعلات متقدمة.
- تحسين تجربة المستخدم.
